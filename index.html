# static-site-boilerplate This is a boilerplate for static HTML, CSS, JS sites ## Set up Complete the following steps to start a new project (NEW-PROJECT-NAME): 1. Clone this repository to your local machine `git clone BOILERPLATE-URL NEW-PROJECTS-NAME` 2. `cd` into the cloned repository 3. Make a fresh start of the git history for this project with `rm -rf .git && git init`
